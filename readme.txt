Description:
Allinpay INTL is a WooCommerce payment gateway plugin that allows customers to make international payments using Allinpay.

Features:
- Accept international payments through Allinpay.
- Secure and reliable payment processing.
- Easy integration with your WooCommerce store.

Installation:

1. Upload the 'allinpay-intl' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to WooCommerce > Settings > Payments, and enable the "Allinpay INTL" payment method.
4. Configure the required settings for the payment gateway, such as your Allinpay account credentials.
5. Save changes, and you're ready to accept international payments with Allinpay.

FAQ:

Q: Can I use this plugin for domestic payments?
A: No, Allinpay INTL is designed specifically for international payments.

Q: How do I get the necessary API credentials for Allinpay?
A: You will need to sign up for an Allinpay merchant account and obtain the API credentials from their platform.
